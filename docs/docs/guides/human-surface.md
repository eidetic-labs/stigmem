# Human Surface (Web UI)

**Audience:** Curators, contributors, and consumers who need a browser UI for managing stigmem facts.

:::info Coming soon
This guide covers the browser-based human surface (B2, ACM-81). Implementation is in progress.
:::

The human surface provides a browser UI for:
- Viewing, searching, and filtering facts by entity, relation, scope, and confidence
- Asserting and retracting facts without API access
- Reviewing and resolving contradictions
- Auditing federation events

When shipped, this guide will cover setup, access control, and common workflows.

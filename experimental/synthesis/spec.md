`experimental/synthesis/` links and for the generated protocol index.

[`features/synthesis/`](../../features/synthesis/). The canonical normative
spec is [`features/synthesis/spec.md`](../../features/synthesis/spec.md).

compatibility location until a future implementation package is explicitly
planned.
